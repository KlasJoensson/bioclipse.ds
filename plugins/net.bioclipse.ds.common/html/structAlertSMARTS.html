Structural Alerts SMARTS

This model evaluates a query compound against a set of structural alerts (also known as toxicophores), implemented as SMARTS [1]. The model makes use of the Chemistry Development Kit for SMARTS querying [2-3]. In case one of the structural alerts identifies a substructure, this is returned as a POSITIVE result for the model.
